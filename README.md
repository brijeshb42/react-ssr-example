## react-ssr-example

An example React project to showcase -

* Server-side rendering
* Hot-reloading
* Typescript

Setup -

1. Clone project

```sh
git clone https://github.com/brijeshb42/react-ssr-example.git
cd react-ssr-example
```

2. Install deps

```sh
yarn
```

3. Start dev server
```sh
yarn start
```
